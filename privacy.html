Privacy Policy 🔒

At Potato Host, we respect your privacy. This policy covers exactly how we manage and secure the information you share with our engine.

1. Information We Collect

We keep data collection minimal. When you create an account, we only collect and store your Email Address. We do not track tracking locations, credit card numbers, or real identities.

2. Data Storage & Security

Your user profile information, authentication logs, and deployment records are stored securely on encrypted database clusters managed via Supabase. Passwords are fully hashed and cannot be read by anyone—including us.

3. Hosted Website Content

The code files you deploy onto Potato Host are public web pages. Do not upload sensitive environment files, private API tokens, or personal databases inside your static HTML structures.

4. Third-Party Routing

We route incoming public network requests through Cloudflare edge systems to prevent DDoS attacks and provide global cache optimization. Cloudflare may process basic server request signatures to block bad bot attacks.
